# Structure Guidelines

Use this file as the local reference for how to structure the PDF created by this skill.

## Main Sections

1. 苦手分析
Identify the root cause and explain the conceptual shift the learner needs.
State the repair target as a concrete action the learner should be able to perform.

2. 覚えるべき公式
List formulas, usage conditions, and variable meanings in bullets.
Include "do not use when" notes when the formula is often confused with another method.

3. ビジュアル解説
Explain the concept visually and then tie the figure back to the formulas.

4. 類似問題
Create one targeted follow-up problem and solve it fully.
Use the same underlying concept as the original problem, and say explicitly what stayed the same.

## Style

- Do not use boxed callouts.
- Prefer headings, bullets, compact tables, and short paragraphs.
- Use bold labels for mini-sections such as 「適用条件」 or 「よくある誤り」.
- Keep each section actionable and review-oriented rather than ornamental.
- Prefer one precise repair target over many loosely related review points.
